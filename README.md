# PULSE/tix — Next.js

Demo event ticket booking app built with Next.js 14 (App Router), Tailwind CSS, Three.js (R3F) and GSAP.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Pages
- `/` — Home with 3D hero + GSAP animations
- `/events` — All events with category filter
- `/events/[eventId]` — Event detail + booking flow

Data source: `data/events.ts` (static).
